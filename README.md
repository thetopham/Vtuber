# NovaAI VTuber Demo

A simple browser-based VTuber project inspired by AI streamers like Neuro-sama.

## Features

- Animated avatar (blink + talking expression)
- Chat input with lightweight AI-like response generation
- Browser text-to-speech for voice output
- Mood switching (chill/hype)

## Run locally

Because this is a static project, you can run it with any local HTTP server.

### Option 1: Python

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.

### Option 2: VS Code Live Server

Open `index.html` with Live Server.

## Notes

This is an original sample character and not a clone of any existing VTuber model.
